#Preference Shooting

  • python

Will Marvel fans love Nike too?
If Nike wants to collaborate with a Movie, which Movie should they choose to make a big profit?
More generally, Is there a relationship between one's brand preference and music/movie tastes?
We actually don't know! However, we thought that choosing or deciding something reflects one's tastes unconsciously.
So, we checked it using Machine Learning.
This project and the result would be useful for a brand or movie/music marketers.
